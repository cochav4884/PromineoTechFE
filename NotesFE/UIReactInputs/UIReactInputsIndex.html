React Inputs

In this lesson, we will explore how to create and manage inputs in a React application. We will cover controlled components, handling form submissions, and managing state with hooks.

Say you have a list of the USA states and as soon as you start typingin the search box, (example: nev) you want the list to start filtering through and only show the states that start with (nev).

Handling Inputs

  • 1. Pick/make a peice of state to connect with the input's value
  • 2. Update the state when the user changes the value of the input
  • 3. Render the input with its value set to the piece of state

Controlled Input

An input that is completely controlled and hooked tightly to React state. This means that the value of the input is always determined by the state, and any changes to the input will update the state accordingly.
