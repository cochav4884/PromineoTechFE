Loops

Let's say we need to do something over and over again until some condition is met. For example:

  1. baking a cake

    recipe calls for 5 cups of flour

  2. place 1 scoup of flour into mixing bowl
  3. repeat
  4. stop when you reach 5 scoups of flour

var cupsOfFlour = 0; while (cupsOfFlour < 5) { console.log('Scooping a cup of flour into the bowl'); cupsOfFlour += 1; console.log('There are ' + cupsOfFlour + ' cups of flour in the bowl'); }

for (var cupsOfFlour = 0; cupsOfFlour < 5; cupsOfFlour++) { console.log('Scooping a cup of flour into the bowl'); console.log('There are ' + cupsOfFlour + ' cups of flour in the bowl'); }

For Loop - prints 0 to 9

for (var i = 0; i < 10; i++) { console.log(i); }

If Statement inside For Loop

for (var i = 0; i < 100; i++) { if (i % 3 == 0) { console.log(i); } }

Do While Loop

let i = 10; do { i++; console.log(i); } while (i < 3);
