Arrays

An array in JavaScript is a container which holds a number of values. You can set a length of an array when it is declared, but you can always add elements to this. Some useful array information and vocabulary is contained here:

  • new Array() or [] tell JavaScript that we are declaring an array
  • element: each item stored in an array is called an element
  • index: each element in an array is found or accessed by its numerical index.
  • arrays are zero-based, so the index values start at 0.
  • the first element in any array is located at index: 0
  • the last element of any array is located at index: arrayName.length - 1
  • additional elements can be added to an array by using: arrayName.push(newValue); Just like any variable in JavaScript, an array must be declared. We declare the array to give it a name.

List and Position of an Array

[a b c d e f] = [0 1 2 3 4 5]

An array list contains a number of items: in this case 6 items are in our array list. The position of the items begin with position 0 and end with position 5. The last item is always the number of items -1.

Vocabulary Terms:

  • Array: An Array is a collection of values accessed through a single variable. The Array Object allows us to store and manipulate multiple values through a variable by using an Index.
  • concat(): A method that combines two or more Arrays and returns a new Array that contains all the elements from the original Arrays. The original Arrays are not modified.
  • Element: Refers to a single item in an Array.
  • filter(): A method that creates a copy of an Array with all the elements that qualify a test implemented by a provided Function.
  • for loop: A for loop is a control flow statement in JavaScript that allows you to execute a block of code repeatedly based on a specified condition. It is often used to iterate over the elements in an Array.
  • for of loop: Allows you to iterate over the values of an iterable object, such as an Array.
  • forEach(): A built-in Array method that allows you to loop through the elements of an Array and execute a provided Function once for each element.
  • Index: A non-negative numerical value that represents the position of an element in an Array. In JavaScript, Arrays are zero-indexed.
  • length: A property that returns the number of elements in the Array.
  • map(): A method that creates a new Array with new elements that result from calling a provided Function on every element in the original Array.
  • pop(): A method that removes the last element from an Array and returns that element.
  • push(): A method that adds one or more elements to the end of an Array and returns the new length of the Array.
  • reduce(): A method that allows you to reduce an Array of values into a single value.
  • reverse(): A method that reverses the order of elements in an Array. It modifies the original Array and returns the Array where the last element is now the first and what was the first element in now the last.
  • shift(): A method that removes the first element from an Array and returns that element. The remaining elements in the Array are then shifted to a lower index.
  • slice(): A method that returns a new Array that contains a portion of the original Array, determined by the method parameters, the starting index (inclusive) and ending index (not inclusive) of the elements to be included in the new Array.
  • sort(): A method that reorganizes the elements of an Array by ascending order based on their assigned Unicode numbers (or code points). The sorted Array is then returned.
  • splice(): A method that allows you to add, remove, and replace elements in an Array at any index. Determined by the parameters passed in to the method.
  • unshift(): A method used to add one or more elements to the beginning of an Array and returns the new length of the modified Array. All other elements in the Array will have their indices increased accordingly.

What is an Array?

An Array is an object data structure in JavaScript that allows us to store and access multiple values of the same data type. Arrays are objects that are resizable and can hold different data types all mixed together, if necessary. Arrays can be created by declaring a variable using the key words var or let and assigning it to an Array literal, which is denoted by square brackets [].
Arrays can also be declared by using an Array constructor with or without the new keyword. Throughout this lesson we will practice using both key words to declare an Array, var and let. Here are different examples of declaring an Array:

Using an Array Literal:

var myArray = []; // Output: [] , an empty Array literal
var myOtherArray = [1, 2, 3]; // Output: [1,2,3]
var myStringArray = ["apple", "banana", "orange"]; // Output: ["apple","banana","orange"]

Using an Array Constructor:

let numsArray = new Array(3); // Output: [ , , ] an empty Array with 3 slots
let myOtherNumsArr = new Array(1, 2, 3); // Output: [1, 2, 3]
let fruitsArr = Array ("apple", "banana", "orange"); // Output: ["apple","banana","orange"]

Sorting Arrays

Sorting an Array means arranging its elements in a particular order. By default, the sort() method sorts elements in ascending order by their Unicode designation. Here is an example of how the sort() method works with an Array of numbers:

// using just the sort() method
let numbers = [3, 25, 4, 12, 5, 2, 6, 11, 21, 5, 1, 15];
numbers.sort();
console.log(numbers); // Output: [1,11,12,15,2,21,25,3,4,5,5,6]

Notice how the numbers are in order but 11 is treated like a 1 and 21 is treated like a 2. Not exactly ascending order but that is how the sort() method works with numbers. However, we can provide a Function to the sort() method to specify a custom sorting order.

// using the sort() method with a Function passed through
let numbers = [3, 25, 4, 12, 5, 2, 6, 11, 21, 5, 1, 15];
numbers.sort((a, b) => a - b);
console.log(numbers); // Output: [1,2,3,4,5,5,6,11,12,15,21,25]

In this example, we use the sort() method with a comparison Function that compares the elements of the Array in pairs and returns a negative value if a should come before b, a positive value if a should come after b, and zero if they are equal. By subtracting b from a, it can be determined which number, a or b, is larger and which one is smaller. It continues through the Array until the elements are sorted in ascending order.

Reverse Array

Reversing an Array means changing the order of its elements so that the first element becomes the last, the second becomes the second to last, and so on. In JavaScript, we can reverse an Array using the reverse() method.

Here is an example of how to reverse an Array of Strings using the reverse() method:
let fruits = ["apple", "banana", "cherry", "date", "elderberry"];
fruits.reverse();
console.log(fruits); // Output: ["elderberry", "date", "cherry", "banana", "apple"]

In this example, we have an Array of fruit that we want to reverse. We use the reverse() method to reverse the order of the elements in the fruits Array. The console.log() statement outputs the reversed Array of fruit.
