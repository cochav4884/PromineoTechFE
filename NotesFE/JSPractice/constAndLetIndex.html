History of JavaScript and ES6:
JavaScript was invented in 1995 by Brendan Eich. JavaScript became an ECMA standard in 1997,
and is referenced by ECMA-262 -- ECMAScript® Language Specification.
Versions of ECMA-262 -- ECMAScript® Language Specification are as follow:
ES1 - ECMAScript 1 in 1997: Original version
ES2 - ECMAScript 2 in 1998: Edits to the Original version
ES3 - ECMAScript 3 in 1999: Added try/catch, switch, do/while, etc.
ES4 - ECMAScript 4: not released
ES5 - ECMAScript 5 in 2009: The first major revision of the JavaScript language
ES6 - ECMAScript 2015: The second major revision of the JavaScript language
Yearly Updates:
ES7 - ECMAScript 2016
ES8 - ECMAScript 2017
ES9 - ECMAScript 2018
ES10 - ECMAScript 2019
ES11 - ECMAScript 2020
ES12 - ECMAScript 2021
Version 13 ES2022 - ECMAScript 2022
