Vocabulary Terms

Comparison Operator

Used to compare two values and returns a value of true or false.

  1. greater than (>)
  2. less than (<)
  3. greater than or equal to (>=)
  4. less than or equal to (<=)
  • console.log(7 > 2);
  • prints: true

    7 is greater than 2

  • console.log(7 < 2);
  • prints: false

    7 is not less than 2

  • console.log(7 >= 2);
  • prints: true

    7 is greater than 2 but not equal to 2, since one of the statements is true the result is true.

  • console.log(7 <= 2);
  • prints: false

    7 is not less than and 7 is not equal to 2, since both statements are false, the result is false.

Equality Operator

An equlity operator is used to compare two vales for equality and returns aboolean value of true or false.

  1. (==)

    The double equals sign is used to check for loose equlity, which means that the two values are equal in value but not necessarily in type.

  2. (===)

    The triple equals sign is used to check fof strict equality, which means that the two values are equal in value and type.

Null

The Null type represents an intentional absence of any object value, and has exactly one value: null. In JavaScript, null is a primitive value and is not an object, but can be used where an object is expected in JavaScript.

Number

A data type that represents both integers and floating-point numbers.

Strict Equality Operator

The Strict Equality Operator, or (===), is a comparison operator that tests if two values are equal, without performing type coercion. If the two values being copared are of different types, the result will be false.

String

A data type that represents a sequence of characters. Strings can be created using

  1. single quotes (' ')
  2. double quotes (" ")

and can be concatenated using the addition (+) operator

Type coercion

The automatic or implicit conversion of a value from one data type to another by the programming language, often in order to perform an operation or comparison.

Undefined

The Undefined type represents a variable that has been declared but has not been assigned a value. It only has one value: undefined.

Variable

A container that holds a value, which can be a

  4. other data type

let

A keyword that is now used primarily in place of var to declare variables in a local scope.

Concatenation

In programming, the addition (+) operator can be used in the additon of numbers and also in the joining of strings. When strings are put together it is call concatenation and the addition (+) operator is then referred to as the concatenation operator.

  1. "Hello" and "World"
  2. var greeting = "Hello";
  3. var subject = "World";
  4. var message = greeting + " " + subject;
  5. console.log(message);
  6. answer: Hello World

Concatenation can get tricky when mixing data types. Adding a string and a number using the addition (+) operator, will result in a string.

  1. "5" + 3
  2. var x = "5";
  3. var y = 3;
  4. console.log(x + y);
  5. result: "53"
  1. 82 + "20"
  2. var x = 82;
  3. var y = "20";
  4. console.log(x + y);
  5. result: "8220"
  1. "1053" + 21
  2. var x = "1053";
  3. var y + 21;
  4. console.log(x + y);
  5. result: "105321"

Similarly, the expression 5 + "3" would result in a string as well & all other reverse as well.

  1. 5 + "3"
  2. var x = 5;
  3. var y = "3";
  4. console.log(x = y);
  5. result: "53"
  1. 82 + "20"
  2. var x = "82";
  3. var y = 20;
  4. console.log(x + y);
  5. result: "8220"
  1. 1053 + "21"
  2. var x = 1053;
  3. var y + "21";
  4. console.log(x + y);
  5. result: "105321"

Similarly, if you concatenate a number and a string such as 3 + "apples", the result is the string "3apples".

When you add an undefined value to a number in JavaScript, the result will be NaN (Not a Number). This is because undefined is not a number and cannot be added to a number.

  1. var a;

    this variable has been declared but not assigned a value, therefore it is undefined.

  2. var b = 4;

    this is a number

  3. console.log(a + b);

    this will result in NaN

  4. result: NaN

Reassign Variables

Variables can be reassigned, or in other words, assigned to different values. For example:

  1. var myVariable = "Hello, world!";
  2. console.log(myVariable);
  3. result: Hello, world!
  4. myVariable = "Goodbye, world!";
  5. console.log(myVariable);
  6. result is now: Goodbye, world!
